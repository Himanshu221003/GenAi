# GenAI-Powered Career Intelligence Platform Ecosystem

A production-grade Full Stack GenAI platform designed to automate resume analysis, ATS optimization, skill-gap detection, and AI-powered interview preparation workflows.

Built using React.js, Node.js, Express.js, MongoDB, JWT Authentication, Gemini AI, and Puppeteer.

---

# 🚀 Features

* Secure JWT Authentication with Token Blacklisting
* AI-powered Resume Analysis & Skill Extraction
* AI-based Skill Gap Detection using Gemini AI
* ATS-Optimized Resume Generation
* AI-generated Interview Questions & Reports
* Dynamic PDF Resume Generation using Puppeteer
* Protected Routes & Role-based Authorization
* Modular Full Stack Architecture with REST APIs

# ⚙️ Installation & Setup

## Clone Repository

```bash
git clone <repository-url>
cd project-folder
```

## Install Dependencies

```bash
# Frontend
cd frontend
npm install

# Backend
cd backend
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the server directory.

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_gemini_api_key
```

---

# ▶️ Run Application

```bash
# Backend
cd backend
npm start

# Frontend
cd frontend
npm run dev
```
